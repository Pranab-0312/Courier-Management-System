# 🚗 Car Rental System

A full-stack web application that allows users to rent cars online with ease. The system provides features for booking vehicles, selecting drivers, and managing car inventory.

## 🔧 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## ✨ Features
- User Registration & Login Authentication
- Car Listing & Availability Check
- Booking System (Based on KM / Day)
- Driver Selection Option
- Admin Panel for Car & Driver Management
- Responsive User Interface

## 🚀 How to Run
1. Install XAMPP / WAMP
2. Place project folder in `htdocs`
3. Import database in phpMyAdmin
4. Start Apache & MySQL
5. Open browser: `http://localhost/project-folder`
